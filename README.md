# SiTix

SiTix is a web app designed to sell tickets.

Access the website here: https://tix.busy-shannon.cloud

## Installation

Use the package manager npm to install modules.

```bash
npm install serve
npm install
npm run build
```

## Usage

```bash
serve -s build
```
